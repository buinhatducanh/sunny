---

## Quy ước đặt tên

```
{Store}_{Phase}_{Rarity}_{Number}
Ví dụ: CAFE_EARLY_COMMON_001
       ELEC_MID_RARE_012
       CLOTH_LATE_EPIC_003
```

**Mã ngành nghề:**
- `SW` = Software Engineering
- `HW` = Hardware Engineering
- `MK` = Marketing
- `GD` = Graphic Design
- `LW` = Lawyer
- `EE` = Electrical Engineer
